# Hackster.io Posting Guide

## Step 1 — Create the project
1. Go to https://www.hackster.io/projects/new
2. Sign in / create a free account
3. **Title**: `Building a Reliable Smart Garage Monitor with a Raspberry Pi, FastAPI, and ntfy`
4. **Tags**: `raspberry pi`, `python`, `fastapi`, `iot`, `home automation`, `gpio`, `garage`
5. **Difficulty**: Intermediate
6. **Time**: 2–4 hours

## Step 2 — Upload images first (Media Library)
Upload all 6 images from the `article_images/` folder before writing. Hackster lets you insert them by clicking the image icon in the editor.

| File | Where to insert |

| `pi-relay-setup.png` | Top of "How the Hardware Works" |

## Step 3 — Fill in the Hackster sections

Hackster structures projects into named sections. Use these:

| Hackster Section | Content to paste |
|---|---|
| **Story** | Everything from "The Problem" through "What I'd Do Differently" |
| **Code** | Link to GitHub: https://github.com/kenigma/smart-garage |
| **Schematics** | The ASCII wiring diagram (from the article) |
| **Components** | Add parts from the Parts List table |

## Step 4 — Components list (copy into the Hackster components section)
- Raspberry Pi 3 Model B+ × 1
- 5V 4-channel relay module × 1
- Magnetic reed switch (NO type) × 1
- Jumper wires (assorted) × 1 set
- Double-sided tape × 1

## Step 5 — Set cover image
Use `pi-relay-setup.png` as the cover/hero image.

## Step 6 — Publish
Click **Publish** when done. Hackster will review and list it publicly within a few hours.
